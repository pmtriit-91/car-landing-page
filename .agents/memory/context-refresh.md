# Context Refresh Protocol

Khi bắt đầu session mới hoặc task lớn:

1. Đọc root `AGENTS.md`.
2. Đọc `routers/skill-activation.md`.
3. Đọc project-specific memory nếu có.
4. Đọc file liên quan trực tiếp đến task.
5. Chỉ sau đó mới code.

Nếu task liên quan phần đã làm trước đây, tìm trong memory trước khi suy luận lại từ đầu.

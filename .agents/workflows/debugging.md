# Debugging Workflow

Khi debug:

1. Reproduce bằng thông tin hiện có.
2. Xác định symptom và boundary.
3. Truy vết data flow / render flow.
4. Sửa nguyên nhân gốc, không chỉ che lỗi.
5. Kiểm tra side effect.

Không đoán lỗi khi chưa đọc file liên quan.
Không rewrite toàn bộ component chỉ để sửa bug nhỏ.

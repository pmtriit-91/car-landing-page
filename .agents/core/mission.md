# Core Mission

Mục tiêu của hệ thống Agent là hỗ trợ phát triển phần mềm ổn định, nhất quán và an toàn trên nhiều dự án.

Nguyên tắc cốt lõi:

- Giữ nguyên hành vi đúng đang tồn tại.
- Không phá cấu trúc project để tối ưu cục bộ.
- Ưu tiên hiểu ngữ cảnh trước khi code.
- Ưu tiên tính nhất quán hơn sự sáng tạo không cần thiết.
- Chỉ áp dụng skill phù hợp với task hiện tại.
- Khi thiếu ngữ cảnh, đọc file liên quan trước; không đoán architecture.

Kết quả mong muốn:

- Code đúng yêu cầu.
- Ít side effect.
- Dễ review.
- Dễ bảo trì.
- Tôn trọng convention hiện có của dự án.

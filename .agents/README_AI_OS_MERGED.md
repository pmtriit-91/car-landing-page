- Existing Vercel composition skills

- Existing git commit excellence skills
- Context-safe project memory template
- Base safety/workflow/reviewer structure

## Important loading rule

Do **not** load the entire Claude Code AI OS pack by default.
Use it as a selective reference layer for planning, code review, security, debugging, testing, research, and memory workflows.

## Recommended Antigravity/Gemini behavior

1. Read `.agents/AGENTS.md` first.
2. Use `.agents/routers/skill-activation.md` for base skills.
3. Use `.agents/routers/advanced-ai-os-activation.md` only when advanced agent behavior is needed.
4. Keep current project files and current project memory as source of truth.
5. Never bring old project context into a new project.
